# قیمت نقره

انس به دلار

`[silvestor] = [silvestor ons dolar]`

انس به ریال

`[silvestor ons rial]`

انس به تومان

`[silvestor ons toman]`

انس به هزار تومان

`[silvestor ons hezar]`

انس به میلیون تومان

`[silvestor ons mil]`


## گرم

گرم به دلار ( ریال / تومان / هزار / ملیون )

`[silvestor gram dolar]`



## مثقال

مثقال به دلار ( ریال / تومان / هزار / ملیون )

`[silvestor mesghal dolar]`


## چند گرم

`[silvestor 10 dolar]`
